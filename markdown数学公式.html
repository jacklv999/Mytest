markdown数学公式语法

markdown支持完整LaTex数学公式语法

1.上标和下标

  • 下标使用符号Y_1表达,公式表达结果为: y1
  • 上标使用 X^{2m}或者X^2, 其中较为复杂的表达式用{}包含, 公式表达结果为: X2m+X2

3.特殊符号

  • 求和符号: 使用\sum 转义表达求和符号,下限符号使用_{m},上限符号使用^\infty, 如: ∑∞m=0
  • 积分符号: \int_0^1转义表达积分符号, 如: ∫10
  • 极限符号: \lim_{变量 \to 表达式} 表达式, 如 lim1→5表达式
  • 向量符号: \vec{a}转义表达向量, 如: a⃗ 
  • 空格符号: markdown支持四种空格\, 、\;、\quad 和 \qquad

4.希腊字母的大小写

希腊字母的大写使用首字母大写\Gamma转义,小写使用首字母小写\gamma转义, 公式表达结果为: Γ 和 γ

5.关于注释

  • 公式注释: 使用\text{内容}转义注释,公式表达结果为: 公式注释
  • 文字颜色: 使用\color{颜色}{文字}转义表达带颜色的文字,公式表达结果为: 文字

6.分数

分数使用\frac{part I}{part II}转义表达,公式表达结果为: (−1)mm!(m+1)

7.公式中的大括号或大分隔符

  • 在配对符号中.如公式中的括号使用\left(表达左括号,使用\right)转义表达向右的括号;如: (x2)
  • 在非配对符号中以\left. 其他 \right符号 或 \left符号 其他 \right. 表非配对的较大符号.如: dudx∣∣x=0
  • 注: 转义表达不需要{ 和 }

8.上划线与下划线

上划线使用\overline{}转义表达,下划线使用\underline{}转义表达,公式表达结果为: a+b+c+d¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯和a+b+c+d−−−−−−−−−−

9.开根号

多次方根使用\sqrt[n]{x},其中n表示根的次数,x表示被开方项,公式表达结果为: 3√n

12.使用HTML语法表达数学公式

  • 上标与下标的表达: 使用标签<sub>和<sup>表达, 表达结果为: text 与text
  • 上划线表达: 使用标签<SPAN style="TEXT-DECORATION: overline">X</SPAN>表达,表达结果为: X
